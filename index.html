Modelos de servicio

Podemos separar la computación en la nube basandonos en su infraestructura, plataforma y aplicaciones.

Software como servicio (SaaS)

Es el modelo que se basa en la distribución de software e el que las aplicaciones se alojan en la nube, junto al servicio y la plataforma (IaaS) y (PaaS) respectivamente

Plataforma como servicio (PaaS)

La plataforma como servicio es un modelo de servicio que ofrecen los desarrolladores para una plataforma completa para desarrollar, ejecutar y gestionar aplicaciones sin llegar a ver la infraestructura subyacente, algunas caracteristicas son:

  • Desarrollo simplificado: PaaS ofrece herramientas de desarrollo simplificadas preconfiguradas que facilitan la creacion de aplicaciones
  • Gestión de infraestructura: Los proveedores Paas se encargan de la gestion de la infraestructura como servidores, almacenamiento y redes asi como el sistema operativo
  • Escalabilidad: PaaS permite un escalado dinámico para manejar los picos de demanda, todo para que los desarrolladores no tengan que realizar ajustes manualmente de los recursos
  • Colaboración: Los entornos de PaaS ofrecen herramientas que facilitan el desarrollo asi como el control de versiones y gestión de proyectos

Algunos ejemplos de PaaS son: Google App Engine que es una plataforma que permite desarrollar como desplegarlas en la infraestructura de google, Microsoft Azure App Service Que es el srvicio de microsoft para crear y alokar aplicaciones web y moviles de forma escalable, Heroku Que es una plataforma que soporta múltiples lenguajes de programación y facilita su implementacion.

Infraestructura como servicio (IaaS)

Este modelo como servicio ofrece al usuario, las capacidades tales como procesamiento, almacenamientos, redes y otros recursos de computacion fundamentales que son importantes para que el que consuma sea capaz de esplegar y ejecutar software, este puede incluir sistemas operativos y aplicaciones, el usuario no gestiona ni controla la infraestructura como servicio de la nube

Algunos ejemplos de infraestructura como servicio (IaaS) son:

  • Servicios de copia de seguridad y recuperacion de archivos y almacenes de datos sin procesar en servidores y equipos de escritorio.
  • Computo como recursos del servidor que ejecutan sistemas basados en la nube, como memoria, procesador, entre otros.
  • Redes de distribucion de contenido (CDN) que son una red de servidores especializados para la distribucion de contenido web y multimedia a los dispositivos que se conecten.
